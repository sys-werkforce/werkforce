# Information Security - five starter task ideas

- Build the first account register: every business account the founder names - email, banking, payments, domain, storage, paid tools - with 2FA status and password uniqueness recorded exactly as the founder reports them, `(not set yet)` wherever the answer is "I don't know"
- Draft the customer-data map: every place customer information lives and who can see each place, unknowns left honestly blank
- Run the founder's first phishing drill: three to five scenario messages built at the desk, never sent anywhere, scored honestly with the missed tells recorded in department memory
- Write the ranked fix list from the first register pass: top three exposures, each fix one action the founder takes at their own keyboard, spending items flagged for the decision queue
- Draft the customer-data handling rules page in plain language - what we store, where it may live, who may see it, what we never do with it - queued for the founder to ratify before it binds anyone
