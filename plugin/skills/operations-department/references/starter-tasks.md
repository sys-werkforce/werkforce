# Operations starter tasks

- Run the first SOP audit across every active department: read each playbook against its memory and board, mark stale candidates, promotion-ready candidates, and missing steps, each with its evidence line
- Build the tool stack checklist from the founder's own list of what the company pays for: name, job, cost with source, renewal date, and a keep/question verdict per tool - any cancel or buy gets queued, never done
- Assemble the first ranked fix-list one-pager: every gap with its evidence and its named owner, smallest fix first, delivered to the outbox
- Draft the smallest SOP proposal for the most-repeated piece of work that no playbook covers - draft-shaped, handed to the owning department to adopt or reject, never written into their playbook from here
- List every cadence the company claims to run - daily, weekly, monthly sessions - and flag the ones the records show never actually happen, with the session lines as evidence
