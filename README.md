# Werkforce - the official plugin marketplace

Your own agentic workforce: a complete company operating system installed as Claude Code skills, run from plain files you own.

## Install

```
claude plugin marketplace add sys-werkforce/werkforce
claude plugin install werkforce@werkforce
```

Then say **"install my werkforce"** in Claude Code to build your HQ.
